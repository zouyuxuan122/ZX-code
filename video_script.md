# ZX-Code 产品介绍视频脚本

## 视频风格参考
模仿 Elsland 宣传视频风格：
- 简约白色背景
- 深色UI界面展示
- 双语字幕（英文标题 + 中文翻译）
- 平滑过渡动画
- 分屏展示（标题卡 + 实际界面）

## 第一幕：AI智能对话（10-30秒）

### 画面3：标题卡
**英文**：Talk naturally. Code effortlessly.
**中文**：自然对话，轻松编码。

### 画面4：AI对话界面展示
**展示内容**：
- 深色主题聊天界面
- 用户输入："帮我写一个快速排序算法"
- AI正在思考（Thinking for 3s...）
- 流式输出代码

**关键特性**：
- 支持多种AI模型（deepseek、claude等）
- 思考级别可调（low/medium/high）
- 代码高亮显示
- Markdown渲染

**UI元素**：
```
┌─────────────────────────────────────┐
│  ZX-Code Agent    [deepseek-v4]    │
├─────────────────────────────────────┤
│  👤 帮我写一个快速排序算法          │
│                                      │
│  🤖 Thinking for 2s...              │
│     正在分析你的需求...              │
│                                      │
│  ```python                          │
│  def quick_sort(arr):               │
│      if len(arr) <= 1:              │
│          return arr                 │
│      pivot = arr[0]                 │
│      left = [x for x in arr[1:]     │
│              if x < pivot]          │
│      right = [x for x in arr[1:]    │
│               if x >= pivot]        │
│      return quick_sort(left)        │
│             + [pivot]               │
│             + quick_sort(right)     │
│  ```                                │
├─────────────────────────────────────┤
│  [📎] [🎤] 输入消息...    [发送]   │
└─────────────────────────────────────┘
```

## 技术规格

### 视频参数
- 分辨率：1920x1080（1080p）
- 帧率：60 FPS
- 时长：约2分30秒
- 格式：MP4 (H.264)

### 音频
- 背景音乐：轻快的电子音乐
- 音效：界面交互音效
- 旁白：可选（中/英文）

### 字幕样式
- 字体：思源黑体 / SF Pro
- 大小：标题 72px，副标题 48px
- 颜色：#000000（黑色）
- 位置：画面中央
